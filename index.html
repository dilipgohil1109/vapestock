Google Drive Sync

Not connected

How it works: All app data is saved as one JSON file in your Google Drive. Every device reads and writes the same file — changes sync within seconds.

What gets synced: Stock quantities · Transaction log · Custom categories · Custom brands · Custom flavors · Deleted built-in flavors

✓ The Google Apps Script code has not changed — no update needed if you already set it up.

Step 1 — One-time Setup
Do this once. Creates the shared file in Google Drive.
1
Go to script.google.com → New Project
2
Delete existing code, paste the code below, then click Save
3
Deploy → New Deployment → Web App → Execute as: Me → Anyone → Deploy
4
Authorize → copy the Web App URL → paste below
Apps Script Code
function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var file = getFile();
  file.setContent(JSON.stringify(data));
  return ContentService.createTextOutput(JSON.stringify({ok:true}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var file = getFile();
  var content = file.getContent();
  return ContentService.createTextOutput(content || '{}')
    .setMimeType(ContentService.MimeType.JSON);
}

function getFile() {
  var folder = DriveApp.getRootFolder();
  var files = folder.getFilesByName('vapestock_data.json');
  if (files.hasNext()) return files.next();
  return folder.createFile('vapestock_data.json', '{}', MimeType.PLAIN_TEXT);
}
